This registration tracks a change made to ISO 639-3 effective
2018-01-23, retiring the code element 'lsg' for Lyons Sign Language as
non-existent. No Preferred-Value is indicated and the existing
Preferred-Value for the extlang is removed. This change affects both the
primary and extended language subtags for this language.

For more information on the ISO 639-3 change, refer to:
http://www-01.sil.org/iso639-3/cr_files/639-3_ChangeRequests_2017_Summary.pdf
